#  Keerthana Durgam – Developer Portfolio



A modern, responsive personal portfolio built to showcase my work, skills, and professional journey as a **Full Stack Developer**.

This portfolio highlights:

* My technical skills and expertise
* Real-world projects I’ve built
* Certifications and continuous learning
* Easy ways to connect with me

Designed with a focus on **minimalism, performance, and user experience**.

---

## ✨ Features

* ⚡ Fast and optimized (Vite-powered)
* 🎯 Clean, modern, and professional UI
* 📱 Fully responsive across all devices
* 🎬 Smooth animations and transitions
* 🧩 Modular and reusable components
* 📌 Section-based navigation:

  * Home
  * About Me
  * Skills
  * Projects
  * Certifications
  * Contact

---

## 🛠️ Tech Stack

* **Frontend:** React + TypeScript
* **Build Tool:** Vite
* **UI Library:** Material UI (MUI)
* **Styling:** Tailwind CSS / Custom styles
* **Icons:** MUI Icons

## 🖥️ Getting Started

### 1. Clone the repository

```bash
git clone git@github.com:keerthanadurgam/keerthanadurgam-portfolio.git
cd know-me
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

App will run at:
http://localhost:5173
